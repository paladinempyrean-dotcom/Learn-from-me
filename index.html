File & Directory Management
ls
List all files and directories in the current folder. Shows hidden files too when used with flags.
Basic
ls -la
Long listing with hidden files. Shows permissions, owner, size, and date for every entry.
Basic
cd <dir>
Change current directory. Use cd .. to go up one level, cd ~ to go home.
Basic
pwd
Print Working Directory — shows the full path of the folder you're currently in.
Basic
cp file dest
Copy a file to a destination. Use cp -r folder dest to copy entire folders recursively.
Basic
mv file dest
Move or rename a file. Works for renaming too: mv old.txt new.txt.
Basic
mkdir <name>
Make a new directory. Use mkdir -p path/to/dir to create nested folders at once.
Basic
rmdir <name>
Remove an empty directory. Will fail if the folder contains any files.
Basic
rm <file>
Delete a file permanently. Use rm -rf folder to delete a folder and all its contents. Be careful!
Dangerous
find . -name "*.py"
Search for files matching a pattern from the current directory downwards recursively.
Advanced
touch file.txt
Create an empty file, or update the timestamp of an existing file without changing its content.
Basic
cat file.txt
Display the entire content of a file to the terminal. Great for small files.
Basic
less file.txt
View large files page by page. Press q to quit, /word to search inside.
Basic
head -n 20 file
Show only the first 20 lines of a file. Change 20 to any number you need.
Basic
tail -f file
Follow a file in real time — great for watching logs as they update live.
Advanced
wc -l file.txt
Count lines, words, or characters in a file. Useful for analyzing file size quickly.
Basic
# Navigate and explore
$ pwd
/data/data/com.termux/files/home
$ mkdir projects && cd projects
$ touch hello.py && ls -la
total 0 -rw-rw-rw- 1 u0_a123 ... hello.py

Package Management
pkg update
Refresh the list of available packages from the repositories. Always run before installing anything.
Essential
pkg upgrade
Upgrade all installed packages to their latest versions. Run after pkg update.
Essential
pkg install <pkg>
Install a new package. Example: pkg install python, pkg install git.
Essential
pkg uninstall <pkg>
Remove an installed package completely from Termux.
Basic
pkg search <name>
Search available packages by name. Shows description and version info.
Basic
pkg list-installed
Show all packages currently installed in your Termux environment.
Basic
apt list --installed
Alternative way to list installed packages using the apt tool with more detail.
Basic
termux-info
Show detailed system info about your Termux installation — version, device, environment.
Basic
dpkg -l | grep pkg
Check if a specific package is installed using dpkg. Useful for verifying installations.
Advanced
pkg show <pkg>
Display detailed info about a package: version, dependencies, description, maintainer.
Basic
# Full setup sequence for a new Termux install
$ pkg update && pkg upgrade
$ pkg install python git curl wget
$ pkg search nodejs
nodejs/stable 20.x.x A JavaScript runtime

Compression & Archiving
tar -cvf arch.tar dir/
Create a .tar archive from a directory. c=create, v=verbose, f=filename.
Basic
tar -xvf arch.tar
Extract a .tar archive. Add -C /path to extract to a specific directory.
Basic
tar -czvf arch.tar.gz dir/
Create a compressed .tar.gz archive using gzip — smaller file size than plain tar.
Basic
tar -xzvf arch.tar.gz
Extract a .tar.gz (gzipped tar) archive to the current directory.
Basic
zip -r out.zip folder/
Compress a folder into a .zip file. Works with files too: zip file.zip a.txt b.txt.
Basic
unzip file.zip
Extract a .zip file. Use unzip -l file.zip to preview contents without extracting.
Basic
gzip file.txt
Compress a single file with gzip. Replaces original with file.txt.gz.
Basic
gunzip file.txt.gz
Decompress a .gz file back to its original form.
Basic
tar -tzvf arch.tar.gz
List contents of a .tar.gz archive without extracting it first.
Advanced

System & Process Monitoring
top
Live process viewer showing CPU and memory usage. Press q to exit, P to sort by CPU.
Basic
htop
Enhanced interactive process manager with colors and mouse support. Install: pkg install htop.
Advanced
ps aux
List all running processes with user, PID, CPU and memory usage, and command name.
Basic
kill <PID>
Terminate a process by its PID. Use kill -9 PID to force-kill a frozen process.
Caution
uptime
Show how long the system has been running and the average CPU load over 1, 5, and 15 minutes.
Basic
free -h
Display RAM usage in human-readable format — total, used, free, and available memory.
Basic
df -h
Show disk space usage for all mounted file systems in human-readable format.
Basic
du -sh folder/
Show the total disk usage of a folder. Use du -ah to see sizes of all files inside.
Basic
pkill -f name
Kill all processes whose command matches the given name. Safer than killing by PID.
Caution
nohup cmd &
Run a command in the background so it keeps running even if you close the session.
Advanced
⚡

Quick Kill: Find a process name with ps aux | grep python, then kill it with kill <PID>. Or just run pkill python to kill all Python processes at once.

Networking & Internet
ping google.com
Test internet connection by sending packets to a host. Shows response time in ms. Press Ctrl+C to stop.
Basic
curl https://url
Fetch data from a URL. Supports GET, POST, headers. Add -o file to save output.
Essential
wget https://url
Download files from the internet. Better than curl for downloading — resumes broken downloads with -c.
Essential
ifconfig
Show network interface information including IP addresses. Install: pkg install net-tools.
Basic
netstat -tuln
List all open network ports and connections. Useful for checking what's listening on which port.
Advanced
ssh user@host
Connect to a remote server via SSH. The most secure way to manage remote Linux machines.
Advanced
scp file user@host:path
Securely copy files to/from a remote server over SSH. Great for transferring large files.
Advanced
nmap -sV host
Scan a host for open ports and services. Install: pkg install nmap. Use responsibly!
Ethical Use
curl -I https://url
Fetch only HTTP headers from a URL — useful for checking server type, response codes, cache headers.
Advanced
curl -X POST -d data url
Send a POST request with data to an API endpoint. Essential for testing REST APIs.
Advanced

User & Permission Basics
whoami
Display the current user's username. In Termux this shows the Android user account.
Basic
chmod 755 file
Change file permissions. 7=owner rwx, 5=group r-x, 5=others r-x. Makes file executable.
Advanced
chmod +x script.sh
Add execute permission to a script file. Required before you can run ./script.sh.
Basic
chown user file
Change the ownership of a file to a specific user. Use chown user:group file for both.
Advanced
passwd
Change the current user's password. Required to set up password for SSH access.
Basic
ls -l file
View the permission bits of a file: -rwxr-xr-x means owner=rwx, group=r-x, others=r-x.
Basic
umask 022
Set default permissions for newly created files. 022 means new files get 644 permissions.
Advanced
# Make a script executable and run it
$ chmod +x myscript.sh
$ ./myscript.sh
# Check permissions
$ ls -l myscript.sh
-rwxr-xr-x 1 user group 1234 Jun 10 myscript.sh

Text Processing & Search
grep "word" file
Search for a word or pattern in a file. Use grep -r "word" . to search all files recursively.
Essential
grep -i "word" file
Case-insensitive search. Finds "Word", "WORD", "word" all at once.
Basic
sed 's/old/new/g' file
Replace all occurrences of "old" with "new" in a file and print the result. Add -i to edit in-place.
Advanced
awk '{print $1}' file
Powerful text processor. $1 = first column, $NF = last column. Great for log analysis.
Advanced
sort file.txt
Sort lines alphabetically. Use sort -n for numeric sort, sort -r for reverse.
Basic
uniq file.txt
Remove duplicate consecutive lines. Use sort file | uniq to remove all duplicates.
Basic
wc -l file
Count lines. wc -w counts words, wc -c counts bytes/characters.
Basic
cut -d',' -f2 file
Extract specific columns from text. Splits by delimiter , and returns column 2. Great for CSVs.
Advanced
echo "text" > file
Write text to a file (overwrites). Use >> to append instead of overwrite.
Basic
nano file.txt
Simple terminal text editor. Press Ctrl+X to exit, Ctrl+S to save. Best for beginners.
Basic
vim file.txt
Powerful text editor. Press i to insert, Esc then :wq to save and quit.
Advanced

Python & pip in Termux
pkg install python
Install Python 3 in Termux. This also installs pip for package management.
Setup
python --version
Check the installed Python version. Termux uses Python 3 by default.
Basic
python script.py
Run a Python script file. Make sure you're in the correct directory first.
Basic
pip install <package>
Install a Python library. Example: pip install requests flask pandas numpy.
Essential
pip list
Show all installed Python packages and their versions.
Basic
pip install -r req.txt
Install all packages listed in a requirements.txt file — great for project setup.
Advanced
python -m venv env
Create a virtual environment to isolate project dependencies from the global Python install.
Advanced
source env/bin/activate
Activate a virtual environment. Your prompt will change to show (env) when active.
Advanced
python -c "code"
Run a single line of Python directly from the command line without creating a file.
Basic
# Quick Python HTTP server
$ python -m http.server 8080
Serving HTTP on 0.0.0.0 port 8080...
# Access from browser: http://localhost:8080

Git Version Control
pkg install git
Install Git in Termux. Essential for version control and downloading GitHub projects.
Setup
git clone <url>
Download a GitHub or GitLab repository to your device. Creates a local copy of the project.
Essential
git init
Initialize a new Git repository in the current directory. Creates the hidden .git folder.
Basic
git status
Show the current state of your working directory — which files are modified, staged, or untracked.
Basic
git add .
Stage all changes for the next commit. Use git add file to stage a specific file.
Basic
git commit -m "msg"
Save staged changes with a descriptive message. Good commit messages explain the why, not the what.
Basic
git push origin main
Upload your commits to the remote repository on GitHub or another service.
Basic
git pull
Fetch and merge the latest changes from the remote repository into your local branch.
Basic
git log --oneline
View commit history in a compact one-line format. Shows hash and message for each commit.
Basic
git branch -a
List all branches — both local and remote. The current branch is marked with an asterisk.
Advanced

Pro Termux Tips & Shortcuts
history
Show the last 500+ commands you've run. Use !123 to re-run command number 123.
Essential
Ctrl + C
Stop a running command immediately. The most important keyboard shortcut in any terminal.
Essential
Ctrl + L
Clear the terminal screen. Same as running the clear command.
Basic
Ctrl + R
Search through command history interactively. Type to filter and press Enter to run.
Advanced
cmd > file.txt
Redirect a command's output to a file. Overwrites the file. Use >> to append.
Advanced
cmd1 | cmd2
Pipe — send output of cmd1 as input to cmd2. Example: ps aux | grep python.
Advanced
termux-setup-storage
Grant Termux access to your phone's internal storage. Run this once after fresh install.
Essential
termux-change-repo
Switch the package repository mirror. Useful if package installs are slow in your region.
Advanced
alias ll='ls -la'
Create a shortcut for a long command. Add aliases to ~/.bashrc to make them permanent.
Advanced
tail -f logfile
Monitor a log file in real time as new lines are appended. Perfect for watching running servers.
Advanced
clear
Clear the terminal screen of all previous output. Keyboard shortcut: Ctrl+L.
Basic
exit
Close the current terminal session. Also works to exit a script, SSH session, or sub-shell.
Basic
